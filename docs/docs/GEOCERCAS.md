# Sistema de Geocercas

Permite definir zonas geográficas para control de actividad.

## Tecnología

- Leaflet
- Leaflet-Geoman

## Flujo

```
Usuario dibuja polígono
  ↓
se guarda en tabla geofences
  ↓
se activa o desactiva
```

## Usos

- control de presencia
- asignaciones
- análisis territorial
